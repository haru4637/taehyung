1. Eclipse for Java and Web developers 설치

  • 구글링
  • 다운로드

2. 아파치-톰캣설치

  • Tomcat 9.0으로 설치
  • 압축 해제
  • bin폴더 밑의 startup.bat으로 실행
  • 설정파일은 conf폴더 밑의 server.xml(포트번호 수정)
  • webapps폴더 밑에 html파일을 두면 서비스 할 수 있는 프로그램(서버 프로그램)
  • JAVA_HOME 또는 JRE_HOME 이라는 이름의 환경 변수가 필요함

3. 프로젝트 생성하기

  • Servers탭 -> 서버 등록 (Tomcat 설치 경로 등록)
  • File - New - Dynamic Web Project
  • 프로젝트 이름 설정
  • 서버 선택
  • src/main/webapp에 파일을 두면 서버의 webapps 밑에서 동작하게 된다
